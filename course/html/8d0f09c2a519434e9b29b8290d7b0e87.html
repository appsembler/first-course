Step 1: Modify the HTML: Add some motivational content!

Any good todo app needs motivational content! Paste the code below into the index.html file; place it below the <div id="app"></div> tag. That section of the file should then look like this:

...
<body>
  <div id="app"></div>
  <blockquote id="motivation" class="motivation">
      Here's to the crazy ones. The misfits. The rebels. The troublemakers.
      The round pegs in the square holes. The ones who see things differently. They're not fond
      of rules. And they have no respect for the status quo. You can quote them, disagree with
      them, glorify or vilify them. About the only thing you can't do is ignore them. Because
      they change things. They push the human race forward. And while some may see them as the
      crazy ones, we see genius. Because the people who are crazy enough to think they can
      change the world, are the ones who do.” – Rob Siltanen
  </blockquote>
  <!-- CSS Styles are 100% optional. but they make it look *much* nicer -->
  <link rel="stylesheet" href="lib/todomvc-common-base.css">
  <link rel="stylesheet" href="lib/todomvc-app.css">
  ...

Then click "File" > "Save All". If you've added motivational HTML snippet properly and saved properly, the page should look like this:

Step 2: Modify the CSS: Style the motivational quote

We should change the style of that motivational quote, though. Open the lib/todomvc-app.css file and add the snippet below to the bottom of it.

blockquote.motivation {
  margin-top: 10px;
  margin-bottom: 10px;
  margin-left: 50px;
  padding-left: 15px;
  border-left: 3px solid #ccc;
}

Step 3: Modify the JavaScript: Disable the motivational quote

It's good to have some motivation when starting a task list, but perhaps your user does not want that motivational quote hanging around. When the user clicks away, we should hide the motivational quote; do that by adding this snippet to your JavaScript file located at lib/todo-app.js:

/* hide the motivational quote when the user clicks away */
window.onload = function(){
  var divToHide = document.getElementById('motivation');
  document.onclick = function(e){
    if(e.target.id !== 'motivation'){
      //element clicked wasn't the div; hide the div
      divToHide.style.display = 'none';
    }
  };
};

Then click "Save All". If you added the JavaScript snippet in the right spot, the motivational quote should be disappear when the user clicks to add a todo item:

The software you can run in Virtual Labs is not just limited to development tools, that's just the image we chose (because it's one of our more popular use-cases.) In case you were wondering, the software you just had a go with is called Theia, an open-source IDE that we're fans of.

Just about anything will run in a Virtual lab, provided it:

  • Is Linux-based (for now...)
  • Runs in a browser and/or can be accessed via an IP and port (though there are ways of running desktop applications!)
  • Can be or already has been Dockerized - if you're not familiar with Docker, you might find this article from OpenSource.com interesting, but beware the rabbit hole!
